Inventory Management System (IMS)

The Inventory Management System (IMS) is a data-driven system designed to help organizations efficiently manage and track inventory levels, monitor stock movement, and improve storage and supply chain operations.

  • Inventory data management for categories, suppliers, discounts, and users
  • Generation of large, realistic datasets using Python and the Faker library
  • Business Intelligence (BI) and data analysis queries
  • Database performance optimization using indexing and batch processing
  • Backup and recovery procedures
  • Cloud deployment support for scalability

Technologies Used: Python 3.x, MySQL, Faker, MySQL Connector

Role: Team Leader (Group Project)

Salon Reservation System (FlexiBook)

FlexiBook Salon is a comprehensive appointment booking system for salons and service-based businesses. It streamlines bookings, provides real-time availability, and offers analytics for business owners.

  • Email confirmations for bookings
  • Admin dashboard and analytics reports
  • Secure JWT authentication
  • Mobile-friendly and responsive design

Technologies Used: React, Node.js, MongoDB, JWT, CSS

Project Type: Group Project
